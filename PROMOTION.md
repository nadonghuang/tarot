# 🔮 Promotion

Ever wanted a tarot reading... in your terminal? 🃏✨

Introducing **tarot** — a beautiful CLI tarot card reader with 78 fully detailed cards, 5 spread types (including Celtic Cross), colored ASCII card art, and zero dependencies.

```bash
npx ./tarot celtic    # Full 10-card reading
npx ./tarot daily     # Today's card
npx ./tarot yesno     # Need a quick answer?
```
